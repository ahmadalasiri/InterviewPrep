### TypeScript: Interface vs Type

One of the most asked TypeScript interview questions! Check out our comprehensive guide:

- [TypeScript Interview Questions](Typescript/00-interview-preparation/01-basic-questions.md#q1-what-is-the-difference-between-interface-and-type-in-typescript)
- [Practical Examples](Typescript/01-basics/interface-vs-type.ts)

**Quick Summary:**

- **Interface**: Better for object shapes, classes, and declaration merging
- **Type**: Required for unions, tuples, primitives, and advanced type operations
