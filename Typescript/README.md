### Interface vs Type Quick Guide

**Use Interface when:**

- Defining object shapes for classes
- Need declaration merging
- Building public APIs
- Object-oriented programming

**Use Type when:**

- Creating unions or tuples
- Aliasing primitives
- Advanced type transformations
- Function types
