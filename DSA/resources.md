## üéØ Study Plans

### 30-Day Plan (Beginner)

**Week 1: Arrays & Strings**

- Two pointers technique
- Sliding window
- String manipulation

**Week 2: Linked Lists & Stacks/Queues**

- Linked list operations
- Stack applications
- Queue implementations

**Week 3: Trees**

- Binary tree traversals
- BST operations
- Tree properties

**Week 4: Sorting & Searching**

- Common sorting algorithms
- Binary search variants
- Time complexity analysis

### 60-Day Plan (Intermediate)

Follow 30-day plan, then:

**Week 5-6: Graphs**

- DFS and BFS
- Topological sort
- Shortest paths

**Week 7-8: Dynamic Programming**

- 1D DP problems
- 2D DP problems
- Common patterns

### 90-Day Plan (Advanced)

Follow 60-day plan, then:

**Week 9-10: Advanced Trees**

- Heaps and priority queues
- Trie implementations
- Segment trees

**Week 11-12: Advanced Algorithms**

- Greedy algorithms
- Backtracking
- Bit manipulation
