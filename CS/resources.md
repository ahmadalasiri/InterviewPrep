# Computer Science Resources

Comprehensive list of resources for learning computer science fundamentals including operating systems, networks, system architecture, and more.

## üìñ Books

### Operating Systems

- **Operating System Concepts** by Silberschatz, Galvin, and Gagne
  - Comprehensive textbook
  - Classic reference
  - Covers all major topics

- **Modern Operating Systems** by Andrew Tanenbaum
  - Detailed explanations
  - Real-world examples
  - Multiple editions available

- **Operating Systems: Three Easy Pieces** by Remzi and Andrea Arpaci-Dusseau
  - Free online book
  - Clear explanations
  - Practical approach

### Computer Networks

- **Computer Networks** by Andrew Tanenbaum
  - Comprehensive coverage
  - Protocol details
  - Network architecture

- **TCP/IP Illustrated** by W. Richard Stevens
  - Deep dive into TCP/IP
  - Protocol internals
  - Practical examples

- **HTTP: The Definitive Guide** by David Gourley
  - HTTP protocol details
  - Web architecture
  - Performance optimization

### System Architecture

- **Computer Organization and Design** by Patterson and Hennessy
  - CPU architecture
  - Memory systems
  - I/O systems

- **Computer Systems: A Programmer's Perspective** by Bryant and O'Hallaron
  - System-level programming
  - Memory organization
  - Performance optimization

### Distributed Systems

- **Designing Data-Intensive Applications** by Martin Kleppmann
  - Distributed systems design
  - Data storage
  - Scalability patterns

- **Distributed Systems: Concepts and Design** by Coulouris, Dollimore, and Kindberg
  - Comprehensive textbook
  - Distributed algorithms
  - Case studies
