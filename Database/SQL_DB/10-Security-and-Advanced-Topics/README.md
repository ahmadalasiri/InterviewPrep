# 10 - Security and Advanced Topics

Protecting data and exploring specialized database features.

## Topics
- **Database Security**: Authentication (Windows vs. SQL), Authorization (RBAC), Roles, and Permissions.
- **Data Encryption**: Transparent Data Encryption (TDE), Always Encrypted, Column-level encryption.
- **SQLCLR (Common Language Runtime)**: Writing database logic in .NET (C#).
- **XML & JSON in SQL**: Handling semi-structured data within a relational database.
- **Audit & Compliance**: Tracking changes and access to sensitive data.

## Senior Engineer Considerations
- Security best practices: Principle of Least Privilege.
- SQL Injection prevention and secure coding.
- Modernizing SQL with JSON support for flexible schemas.
- When to use SQLCLR vs. native T-SQL vs. application logic.
