# 08 - Administration and Maintenance

Operational aspects of managing a healthy and reliable database system.

## Topics
- **DB Constraints & Rules**: Primary keys, Foreign keys, Unique, Check, Default.
- **Backups & Restores**: Full, Differential, Transaction Log backups. Recovery models.
- **SQL Server Agent Jobs**: Automation of maintenance tasks and alerts.
- **Database Mail**: Configuring alerts and notifications.
- **Maintenance Plans**: Index rebuilds, statistics updates, integrity checks.

## Senior Engineer Considerations
- Disaster Recovery (DR) vs. High Availability (HA) strategies.
- Backup validation and RPO/RTO (Recovery Point/Time Objectives).
- Automated monitoring and alerting for database health.
- Infrastructure as Code (IaC) for database provisioning.
